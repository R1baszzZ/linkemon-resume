# 🎮 Linkemon Resume

Welcome to **Linkemon**, my interactive resume built with **React + Vite**.

Instead of a traditional portfolio website, I wanted to build something memorable — a small interactive experience inspired by **Pokémon battle screens**.

When you open the site, you're presented with a battle interface where you can choose which "Linkemon" to catch.

Each one represents part of my professional world.

## ⚡ Features

* Keyboard navigation (← → or A / D)
* Pokémon-style selection UI
* Capture animation with **3 shakes**
* Flash transition before navigation
* Links to my professional platforms

## 🎯 Linkemon Targets

| Linkemon     | What it does                  |
| ------------ | ----------------------------- |
| **GitHub**   | Opens my coding repositories  |
| **LinkedIn** | Opens my professional profile |
| **About**    | Scrolls to my personal story  |

## 🚀 Running the Project

Clone the repository:

```bash
git clone https://github.com/R1baszzZ/linkemon-resume.git
cd linkemon-resume
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:5173
```

## 🎮 Controls

| Key   | Action                      |
| ----- | --------------------------- |
| ← / A | Move selection left         |
| → / D | Move selection right        |
| Enter | Catch the selected Linkemon |
